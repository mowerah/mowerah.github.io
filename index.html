Skills

Below are some of the Front End languages which I am well versed in.

HTML

HTML is the standard markup language for creating web pages. It describes the structure, specifies the semantics for your data elements and groups them in logical sets.

CSS

CSS allows you to style and control the layout of your web pages all at once. You can make your website responsive for different devices (mobile phones, tablets and desktops). You can also control things like the background color, font-sizes, how elements are positioned and much more.

JavaScript

JavaScript is a popular programming language for the web. JavaScript makes websites interactive by updating and changing both HTML and CSS.

React

React is a JavaScript UI (User Interface) library that is created by Facebook. It is a powerfull tool for building UI components.

Bootstrap 4

Bootstrap is a very popular css framework. It is used for developing responsive and mobile-first websites.
